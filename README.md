# Webhajo

코로나로 인해 사회성 부족을 겪고 있는 아이들을 위한 사회성 발달 스토리 텔링 게임 웹사이트입니다.
<br>


## 기능
 - 로그인    
 - 회원가입    
 - 게시판 작성, 수정, 삭제    
 - 회원 정보(마이 페이지)    
 - 회원 관리 페이지(선생님 페이지)    
 - 게임    
 - 피드백
